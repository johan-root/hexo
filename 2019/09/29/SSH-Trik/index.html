SSH

Johan
2019-09-29

SSH

SSH, atau Secure Shell, merupakan protokol administrasi remote yang memperbolehkan pengguna untuk mengontrol dan memodifikasi server remote mereka melalui internet.

Contoh Penggunaan SSH:

ssh [user]@[host] = :~$ ssh root@117.53.x.x

Cara Kerja SSH

Command SSH key memberi perintah pada sistem Anda bahwa Anda ingin membuka Koneksi Secure Shell yang dienkripsi. [user] merujuk pada akun yang ingin Anda akses. Misalnya, Anda ingin mengakses pengguna root, yang pada dasarnya sama untuk administrator sistem dengan hak lengkap untuk memodifikasi apa pun pada sistem. [host] merujuk pada komputer yang ingin Anda akses. Yang biasanya ini adalah alamat IP (misalnya, 117.53.x.x)

Generate Key

Tujuan dari melakukan generate key adalah agar kita dapat Public Key dan Private Key. Untuk melakukan generate key dapat menggunakan command berikut:

:~$ ssh-keygen

Hasil dari generate key tersebut akan disimpan di directory
:~$ ~/.ssh/, pada directory :~$ ~/.ssh/ akan terdapat file id_rsa.pub dan id_rsa.

  • id_rsa.pub : Merupakan File Public Key
  • id_rsa : Merupakan File Private Key

Public Key Akses

Syarat agar kita dapat akses kedalam server host menggunakan Public Key adalah Server host harus memiliki Public Key client.

Perintah ssh-copy-id digunakan untuk memberikan/mengcopy Public Key yang kita miliki pada Server Host yang akan kita akses.

Contoh Penggunaan:

:~$ ssh-copy-id root@117.53.x.x
